StatefulWithHelpers

protocol StatefulWithHelpers: Stateful

Provides access to state property via functions.

  • currentState() Extension method

    Provides read-only access to the state property value from Stateful protocol.

    Use as follows:

    class MyView: UIView, StatefulWithHelpers { }

let aView = MyView()

let crrent = aView.currentState()

    Declaration

    Swift

    func currentState() -> State?

    Return Value

    Current state as instance of State without conversion to a specific state type, if it’s set, or nil otherwise.

  • update(_:mutation:) Extension method

    Allows to mutate current state, if it’s of expected type.

    Use as follows:

    class MyView: UIView, StatefulWithHelpers
{
   struct Disabled: State
   {
       var opacity: Float
   }
}

let aView = MyView()

try? aView.update(MyView.Disabled.self){

   $0.opacity += 0.1
}

    Throws

    Errors.WrongState if current state is NOT of expected type.

    Declaration

    Swift

    func update<ExpectedState: State>(
        _ expected: ExpectedState.Type,
        mutation: (inout ExpectedState) -> Void
        ) throws

    Parameters

    expected

    Expected current state type.
    mutation

    Closure that takes as inout input parameter an instance of expected type. This closure will be used to mutate currents state, current state will be passed into it to perform mutations.
  • set(_:) Extension method

    Allows to set current state to newState.

    Use as follows:

    class MyView: UIView, StatefulWithHelpers
{
    struct Disabled: State
    {
        var opacity: Float
    }
}

let aView = MyView()

aView.set(MyView.Disabled(opacity: 0.3))

    Declaration

    Swift

    func set<Input: State>(_ newState: Input?)
